

# BiliVideoPlayer

## 项目简介

本项目是基于 Qt6/C++17 开发的仿哔哩哔哩桌面端视频客户端，采用单例模式 + 集中式数据中心（DataCenter）实现 MVC 分层架构，将 UI 展示、业务逻辑与网络通信解耦。

### 技术栈

- **客户端**：C++17、Qt6（Widgets / Network）、mpv C API、FFmpeg、CMake、QSS、JSON 序列化
- **服务端**：MockServer（用于测试）

### 核心功能

- HLS 流媒体播放
- 实时弹幕系统
- 多角色权限管理（用户、管理员）
- 视频上传与审核
- 用户关注与个人中心

## 构建与运行

### 开发环境

- Qt Creator 6.7.3
- MinGW_64
- C++17 编译器

### 项目结构

```
client/
├── BiliVideoPlayer/     # 客户端主程序
├── MockServer/          # 模拟服务器（用于测试）
└── MyPlayerQt/          # 简易播放器示例
```

### 配置说明

构建完成后，在 `build/*/config.ini` 文件中配置服务器 IP 和端口：

```ini
[server]
ip=127.0.0.1
port=8080
```
