# BiliVideoPlayer（仿哔哩哔哩桌面端视频播放器）- 自主开发

## 项目简介
基于 Qt6 开发的桌面端视频播放与内容管理客户端，采用单例模式 + 集中式数据中心 DataCenter 的方式组织业务数据，并通过信号槽把页面 UI、网络请求和状态刷新解耦。项目内置主页视频浏览、视频播放、弹幕展示与发送、用户登录注册、个人信息管理、视频上传、后台审核与管理员管理等功能，并提供 MockServer 进行联调测试。

## 开发环境与技术栈
C++11、Qt6、QtDesigner 界面设计、QSS 样式配置、自定义控件、FFmpeg、SDL、Qt Network、Qt Json、单例模式、异步 HTTP 通信、MockServer。

## 项目特点
- 播放与控制：封装基于 FFmpeg + SDL 的 FFmpegPlayer 播放模块，支持本地视频文件和 HLS/m3u8 地址播放，提供播放/暂停、进度拖动、音量调节、倍速播放、播放时间同步，以及弹幕同步显示与发送；上传页还会调用 FFmpeg 获取视频总时长，并通过 FFmpeg 命令行提取视频首帧作为封面图。
- 主页视频展示：通过 DataCenter 请求首页视频列表，支持分类、标签和关键词搜索分页浏览，并在页面切换时刷新列表数据。
- 个人信息界面：支持查看当前用户和其他用户主页信息，修改头像、昵称和密码，展示关注数、粉丝数、点赞数和播放数；对其他用户可进行关注/取消关注。
- 账户与登录：支持邮箱验证码登录/注册、账号密码登录、临时用户登录和会话登录，使用 sessionId 作为统一会话标识，并对邮箱和验证码输入做了正则校验与错误提示。
- 视频上传：支持选择本地视频上传，上传流程会先提交视频文件，再上传封面图和视频描述信息；上传页支持选择分类和标签，并限制最多选择 5 个标签。
- 后台管理页面：管理员系统页支持按用户 ID 或视频状态查询视频，支持分页查看、通过/驳回审核、上架/下架视频，并可执行新增、编辑、删除管理员等操作。
- 数据与通信：NetClient 统一封装异步 HTTP 请求，DataCenter 负责请求分发、数据缓存和结果信号转发，界面组件通过信号槽接收结果并更新 UI。
